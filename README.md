# OIT Drone Project
OITドローンプロジェクトのサイトです．
DJITelloPyのライブリーを使用してTelloを操作しましょう．
## DJITelloPy
### pipでインストール
以下のコマンドでDJITelloPyをインストールすることができます．

DJItelloPyのページ
[DJItelloPy](https://github.com/damiafuentes/DJITelloPy)
```bash
pip install djitellopy
```
